# React Would You Rather App Project

This is the Would You Rather App project to practice React, Redux, and Testing.

## Technologies used

React, Redux, React Router, Semantic-ui, Jest, Enzyme, and Cypress are used for this project.

## Install and Launch the project

- make sure you have node installed.
- clone this repo
- install all project dependencies with `npm install`
- start the development server with `npm start`

## Run unit tests

- `npm run test`

## Run cypress integration tests

- `npm run cypress:open`
